## What is Deno

Deno is simple, modern and secure javascript and typescript that uses Chrome's v8 engine and is build in Rust programming language.

1. Released on 13th of May 2020
2. The creator of Deno is Ryan Dahl

Deno is project that requires every node developer attention.

## Run Deno javascript or typescript code

```
deno run index.js
deno run index.ts
```

### Deno Features

1. Built in typescript support.
2. Import from URL
3. Es6 modules import syntax = no commonjs
4. top Level await
5. More compatible with browser, fetch, window
6. Security flags.
7. Ships single executable files.
8. build-in code formatting support
9. Build-in testing support.

### What is Typescript

TypeScript is typed superset of Javascript that compiles to plain Javascript.

##### Typescript Advantages

1. Add types to Javascript.
2. Full Oop support: inheritance, abstraction, interface
3. Generics, Enums, Es6 modules.
4. No learning Curve
5. Better autocompletion and tooling.
6. You can write future javascript code.
7. Catch errors before execution.

### How to execute Typescript

1. Install Typescript compiler - npm install -t typescript
2. Create a \*ts file
3. Write typescript code
4. compile into \*.js file
5. run the \*js file
