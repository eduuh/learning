# Automatically Deployment

A script to automaticall deploy **node or a dotnet site**, to two environments **staging** and **deployment**.

The setup script will create separate folders for deployment. **stag** and **prod**, clone the github code. And copy the code in each folder. Depending on the folder a git checkout is done to the repective branch.
